# Background Images Setup Guide

## Free Image Sources

### Recommended Sites (No Attribution Required)
1. **Unsplash** - https://unsplash.com
   - Search: "vacation", "travel", "beach", "mountains", "productivity"
   
2. **Pexels** - https://pexels.com
   - Search: "holiday", "workspace", "planning", "adventure"
   
3. **Pixabay** - https://pixabay.com
   - Search: "travel destination", "office desk", "calendar"

## Recommended Images for Holiday Tasker

### 1. Landing Page Hero
- **Theme**: Inspiring travel/productivity blend
- **Keywords**: "tropical beach sunset", "mountain adventure", "travel planning"
- **Dimensions**: 1920x1080 or larger

### 2. Login/Signup Pages
- **Theme**: Calm, professional, inviting
- **Keywords**: "minimal workspace", "clean desk", "modern office"
- **Dimensions**: 1920x1080

### 3. Dashboard Background
- **Theme**: Subtle, non-distracting
- **Keywords**: "abstract gradient", "soft texture", "minimal pattern"
- **Dimensions**: 1920x1080

## How to Add Images

1. Download images from the sources above
2. Save them in `src/assets/` folder with descriptive names:
   - `hero-bg.jpg` (already exists)
   - `login-bg.jpg`
   - `signup-bg.jpg`
   - `dashboard-bg.jpg`
   - `tasks-bg.jpg`

3. Images will be automatically imported in the components
